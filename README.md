# Mechanical Engineering Portfolio

This is a lightweight static portfolio site designed for GitHub Pages.

## Add your images

Create this folder:

`assets/images/`

Then add your project photos using these filenames:

- `profile-photo.jpg`
- `project-01-01.jpg`
- `project-01-02.jpg`
- `project-01-03.jpg`
- `project-01-04.jpg`
- `project-01-05.jpg`
- `project-01-06.jpg`

The header profile image and gallery in [`index.html`](/Users/artemgomov/Downloads/Codex/Portfolio Website/index.html) are already wired to those image paths.

## Customize

Update these placeholders before publishing:

- Your name in [`index.html`](/Users/artemgomov/Downloads/Codex/Portfolio Website/index.html)
- Email, GitHub, and LinkedIn links in [`index.html`](/Users/artemgomov/Downloads/Codex/Portfolio Website/index.html)
- Project titles and descriptions in [`index.html`](/Users/artemgomov/Downloads/Codex/Portfolio Website/index.html)

## Publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload these files to the repository root.
3. In GitHub, open `Settings` > `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the main branch and the root folder.
6. Save, then wait for GitHub Pages to publish your site.
